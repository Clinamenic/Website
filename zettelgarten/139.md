> If what's being funded is a big, complicated project like building a hospital, then rigorous transactional analysis is warranted, but there is a cost to centering those with an expertise in such analysis over those who understand the communities where a project takes place. The more relational a project becomes, the less helpful this kind of centralized expertise becomes.
